# 令和ぱずる

日本の元号を揃えて遊ぶ落ちゲー。
左右フリックで移動。タップで回転。
揃った元号の年数が長いほど高得点。
令和はスペシャルポイント1万点。
上まで積み上がるとゲームオーバー。
